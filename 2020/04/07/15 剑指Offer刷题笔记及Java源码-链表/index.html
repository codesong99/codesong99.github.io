03 从尾到头打印链表

输入一个链表，按链表从尾到头的顺序返回一个ArrayList。

import java.util.ArrayList;
import java.util.Stack;
public class Solution {
    public ArrayList<Integer> printListFromTailToHead(ListNode listNode) {
        Stack<Integer> stack = new Stack<>();
        while (listNode != null) {
            stack.push(listNode.val);
            listNode = listNode.next;
        }

        ArrayList result = new ArrayList<Integer>();
        while (!stack.isEmpty()) {
            result.add(stack.pop());
        }

        return result;        
    }
}

13 链表中倒数第K个结点

输入一个链表，输出该链表中倒数第k个结点。

//使用两个指针，front领先behind K-1个结点
//front到尾结点时，behind即为结果
public class Solution {
    public ListNode FindKthToTail(ListNode head,int k) {
        if (head == null || k == 0)
            return null;
        ListNode front = head;
        ListNode behind = head;

        for(int i = 1; i < k; i++) {
            if (front.next == null) 
                return null;
            front = front.next;

        }

        while (front.next != null) {
            front = front.next;
            behind = behind.next;
        }
        return behind;
    }
}

14 反转链表

输入一个链表，反转链表后，输出新链表的表头。

public class Solution {
    public ListNode ReverseList(ListNode head) {
        if (head == null || head.next == null) 
            return head;

        ListNode newHead = head;
        ListNode pre = null;
        ListNode temp = null;
        while(newHead != null) {
            pre = newHead;
            newHead = newHead.next;
            pre.next = temp;
            temp = pre;
        }
        return pre;
    }
}

15 合并两个排序的链表

输入两个单调递增的链表，输出两个链表合成后的链表，当然我们需要合成后的链表满足单调不减规则。

// 使用递归
public class Solution {
    public ListNode Merge(ListNode list1,ListNode list2) {
        if (list1 == null)
            return list2;
        if (list2 == null)
            return list1;

        //哪个头结点小，哪个作为结果的头结点
        if (list1.val >= list2.val) {
            list2.next = Merge(list1, list2.next);
            return list2;
        } else {
            list1.next = Merge(list1.next, list2);
            return list1;
        }

    }
}

24 复杂链表的复制

输入一个复杂链表（每个节点中有节点值，以及两个指针，一个指向下一个节点，另一个特殊指针指向任意一个节点），返回结果为复制后复杂链表的head。（注意，输出结果中请不要返回参数中的节点引用，否则判题程序会直接返回空）

//不需要特殊技巧，一个结点一个结点地next就可以了
//正确理解RandomListNode数据结构
//复制的意思是新建结点，不能返回已有的参数结点
public class Solution {
    public RandomListNode Clone(RandomListNode pHead)
    {
        if (pHead == null)
            return null;
        RandomListNode head = new RandomListNode(pHead.label);
        RandomListNode result = head;

        if(pHead.random != null) 
            head.random = new RandomListNode(pHead.random.label);

        while(pHead.next != null) {
            pHead = pHead.next;
            head.next = new RandomListNode(pHead.label);
            head = head.next;
            if (pHead.random != null)
                head.random = new RandomListNode(pHead.random.label);

        }
        return result;
    }
}

25 二叉搜索树与双向链表

输入一棵二叉搜索树，将该二叉搜索树转换成一个排序的双向链表。要求不能创建任何新的结点，只能调整树中结点指针的指向。

//参考博客https://blog.csdn.net/jiangjiang_jian/article/details/81637574

//将二叉搜索树中序遍历
public class Solution {
    TreeNode head = null;    //头结点
    TreeNode temp = null;    //移动的结点
    public TreeNode Convert(TreeNode pRootOfTree) {
        InOrder(pRootOfTree);
        return head;
    }

    //中序遍历
    public void InOrder(TreeNode pRoot) {
        if (pRoot == null)
            return;

        InOrder(pRoot.left);

        if (head == null) {    //仅使用一次，即head为中序遍历的第一个结点时
            head = pRoot;
            temp = pRoot;
        } else {
            temp.right = pRoot;
            pRoot.left = temp;
            temp = pRoot;
        }

        InOrder(pRoot.right);
    }
}

35 两个链表的第一个公共结点

输入两个链表，找出它们的第一个公共结点。（注意因为传入数据是链表，所以错误测试数据的提示是用其他方式显示的，保证传入数据是正确的）

//齐头并进
public class Solution {
    public ListNode FindFirstCommonNode(ListNode pHead1, ListNode pHead2) {
        if (pHead1 == null || pHead2 == null)
            return null;

        //求两个链表的长度
        ListNode node1 = pHead1;
        int len1 = 0;
        while(node1 != null) {
            node1 = node1.next;
            len1 ++;
        }

        ListNode node2 = pHead2;
        int len2 = 0;
        while(node2 != null) {
            node2 = node2.next;
            len2++;
        }

        //较长的链表先移动
        int diff = len1 - len2;
        if (diff >= 0) {
            while (diff >0 ) {
                pHead1 = pHead1.next;
                diff --;
            }
        } else {
            while (diff < 0) {
                pHead2 = pHead2.next;
                diff ++;
            }
        }

        //齐头并进
        while(pHead1 != null && pHead1.val != pHead2.val) {
            pHead1 = pHead1.next;
            pHead2 = pHead2.next;
        }
        return pHead1;
    }
}

53 链表中环的入口结点

给一个链表，若其中包含环，请找出该链表的环的入口结点，否则，输出null。

// 如果有环，快慢指针一定会相遇
// 相遇点出发、头结点出发 的两个指针，会在环入口处相遇
public class Solution {

    public ListNode EntryNodeOfLoop(ListNode pHead)
    {    
        ListNode fast = pHead;
        ListNode slow = pHead;

        while(fast != null && fast.next != null) {
            fast = fast.next.next;
            slow = slow.next;
            if (fast == slow)    //有环，相遇
                break;
        }        
        if(fast == null || fast.next == null)     //走到头了，无环
            return null;

        fast = pHead;
        while (fast != slow) {
            fast = fast.next;
            slow = slow.next;
        }
        return fast;

    }
}

55 删除链表中重复的结点

在一个排序的链表中，存在重复的结点，请删除该链表中重复的结点，重复的结点不保留，返回链表头指针。 例如，链表1->2->3->3->4->4->5 处理后为 1->2->5

// 此题关键在于新建一个空节点，指向头结点
public class Solution {
    public ListNode deleteDuplication(ListNode pHead){
        if (pHead == null)
            return null;
        ListNode result = new ListNode(0);
        result.next = pHead;

        ListNode pre = result;
        ListNode cur = result.next;
        while(cur != null) {
            if(cur.next != null && cur.val == cur.next.val) {
                while (cur.next != null && cur.val == cur.next.val)
                    cur = cur.next;
                pre.next = cur.next;
                cur = cur.next;
            }
            else {
                pre = pre.next;
                cur = cur.next;
            }
        }

        return result.next;
    }
}
